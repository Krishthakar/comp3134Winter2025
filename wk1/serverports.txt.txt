1. What is the definition of a server port?
Ans:- A server port is a communication endpoint used by software to identify specific processes or services on a networked system. Ports allow multiple applications to use the same network interface simultaneously without interference.

2. What is the server port range?
Ans:- The server port range is from 0 to 65535. These ports are divided into three categories:

Well-known ports: 0–1023
Registered ports: 1024–49151
Dynamic/private ports: 49152–65535

3. How many bits does the number above represent?
Ans:- The port number is represented by 16 bits, as it ranges from 0 to 65535 (2^16).

4. State 10 common server ports.
Ans:- Port 20: FTP (Data Transfer)
Port 21: FTP (Control)
Port 22: SSH
Port 23: Telnet
Port 25: SMTP (Email)
Port 53: DNS
Port 80: HTTP (Web Traffic)
Port 110: POP3 (Email)
Port 143: IMAP (Email)
Port 443: HTTPS (Secure Web Traffic)
